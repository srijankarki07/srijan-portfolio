## Running Locally

Clone the project

```bash
https://github.com/srijankarki07/srijan-portfolio.git
```

Go to the project directory

```bash
cd srijan-portfolio
```

Remove remote origin

```bash
git remote remove origin
```

Install dependencies

```bash
pnpm install
```

Start the server

```bash
pnpm run dev
```
